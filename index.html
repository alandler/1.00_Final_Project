Raspberry Pi IoT Sensors

The SPS30 Particulate Matter Sensor can detect particle diameter in four ranges: PM1.0 (0.3μm-1.0μm), PM2.5 (1.0μm-2.5μm), PM4 (2.5μm-4.0μm) and PM10 (4.0μm-10.0μm).


Seamless Data Collection

Two sensor stations have been built - one for test in the kitchen and the other for control in a bedroom. Data is measured every minute, stored in the microSD card and uploaded to ThingSpeak cloud website
